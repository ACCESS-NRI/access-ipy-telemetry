============
Installation
============


Stable release
--------------

To install access-ipy-telemetry, run this command in your terminal:

.. code-block:: console

    $ pip install access_ipy_telemetry

This is the preferred method to install access-ipy-telemetry, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for access-ipy-telemetry can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/charles-turner-1/access_ipy_telemetry

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/charles-turner-1/access_ipy_telemetry/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/charles-turner-1/access_ipy_telemetry
.. _tarball: https://github.com/charles-turner-1/access_ipy_telemetry/tarball/master
